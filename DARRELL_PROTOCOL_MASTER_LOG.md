### 4. AI Tool Integration Protocol
**When using multiple AI systems (ChatGPT, NotebookLM, Claude):**

- Check for recursive feedback loops (one AI's error propagating through another)

- Document which system analyzed which materials
- **Audit Loop**: Submit outputs back to AI for consistency check, then refine
