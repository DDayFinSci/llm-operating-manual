# Universal Naming Convention

Format
`<YYYYMMDD>__<Project>__<Unit>__<Status>__<vX>__<agent>.<ext>`

Examples
- 20251108__CFA-L1__econ-elasticities__EXTRACT__v1__claude.md
- 20251108__WebBank-CPR__ratio-covenants__SYNTH__v3__gpt.md
- 20251109__LifeOS__monthly-budget__FINAL__v1__human.xlsx

Notes
- No spaces; use hyphens.
- Increment vX only when meaning changes.
- Agent tag = gpt | claude | nblm | human.
